# 延安丁峰窑洞养老幸福家园项目申报

**申报单位**：延安丁峰窑洞养老服务有限公司

**项目地址**：延安市宝塔区南市街道高家园则3号院

**申报日期**：2026年5月14日

---

## 一、单位基本情况

延安丁峰窑洞养老服务有限公司成立于2025年，注册资本10万元，是专注本土化普惠养老服务的专业机构。公司坐落于南市街道高家园则3号院，地处城中村核心区位，毗邻延安市人民医院、宝塔区人民医院，老人就医便捷，交通路网四通八达，长期与专业医疗卫生机构签订合作协议，常态化开展老年健康体检、慢病管理、康复指导等服务。

机构占地面积1200平方米，拥有陕北特色康养窑洞18孔，科学规划功能分区：自理生活区窑洞10孔，配套适老化传统大炕；失能护理区窑洞8孔，配置专业护理床位24张，全屋布设烟感器、一键呼叫器、摔倒报警器、生命体征监测仪等智慧养老设备，可同时容纳54位长者入住休养。院内全域完成专业适老化改造，铺设防滑防撞地面、配齐无障碍通行设施，每孔窑洞配备独立卫生间、24小时恒温热水系统及洗浴专用座椅；采用燃气集中供暖+全屋地暖双供暖模式，环境恒温舒适；同步打造生态康养场景，配套自有菜园、休闲廊架、八角亭，宜居性与安全性兼具。

公司依托二十年家政行业底蕴、自有职业技能培训学校资源，创新打造公益+市场+社区合伙人运营模式，构建养老服务全链条体系；免费为辖区居民开展养老护理技能培训、助力居民就业，严格落实养老消费券政策，针对独居失能老人推出居家服务半价券抵扣福利，目前已兜底保障8户特困老人基本养老服务，切实履行民生服务责任。同时独创"冬南夏北"候鸟式康养模式，在海南三亚湾设立分公司，实现长者夏季延安避暑、冬季三亚越冬的四季康养服务。

---

## 二、项目建设背景及依据

为积极应对人口老龄化，全面落实2026年宝塔区民生实事项目部署要求，加快推进街道、社区养老幸福家园全覆盖，完善城乡普惠养老服务体系，构建"15分钟养老服务圈"，本机构依托自有窑洞存量资源，采用改扩建提升模式，打造具有陕北本土特色的标准化养老幸福家园，重点保障高龄、空巢、独居、失能、特困及术后康复老年群体养老刚需，补齐社区养老服务短板，提升辖区养老服务保障水平。

---

## 三、项目建设主要内容

1. **适老化改造工程**：完成窑洞全域防滑地面铺设、无障碍扶手安装、卫浴适老化升级、无高差通行改造、柔和照明及醒目功能标识安装；升级燃气集中供暖与全屋地暖系统，优化供暖管网，保障冬季供暖安全稳定。
2. **设施设备配置**：实现院内监控无死角全覆盖，采购专业护理床、适老化家具、智慧安防设备、健康监测器械、老年助餐设施、康复理疗器材、文娱活动设备及消防应急器材，完善各功能区硬件配套。
3. **服务体系搭建**：严格落实日间照料、老年助餐、休闲娱乐、上门探访四项基础服务，拓展助医、助浴、助洁、助行、助急、康复理疗、短期托养、喘息照护、候鸟旅居等特色服务，构建全周期养老服务体系。

---

## 四、项目投资概况

本项目总投资120万元，资金全部专款专用，用于项目建设及设施设备购置、相关手续办理，具体支出明细：

1. 窑洞适老化装修改造费用：62万元；
2. 燃气供暖及全屋地暖升级费用：26万元；
3. 专业护理床及康养家具采购费用：16万元；
4. 智慧安防、健康监测设备采购费用：9万元；
5. 院内环境提升及文娱配套设施费用：5万元；
6. 房屋安全鉴定报告费用：2万元。

---

## 五、项目面临实际困难

项目已完成前期规划、功能分区设计等筹备工作，因企业自身资金实力有限，120万元项目建设资金存在大额缺口，适老化改造、供暖升级、设施采购、房屋安全鉴定等关键环节推进滞后，无法按期完工；项目正式运营后，护理人员薪酬、物资采购、设备维保、日常运维、特困老人公益服务等固定开支庞大，仅依靠企业自营收入，难以维持长效稳定运营，亟需政府资金扶持保障项目落地见效。

## 七、项目社会效益

本项目建成运营后，将有效完善南市街道社区养老服务网络，兜底特殊困难老年群体养老需求，切实解决辖区老人养老难、照料难问题；充分盘活沟道治理闲置资源，联动政府、企业、社区三方力量，打造本土化、可持续的社区养老运营样板，进一步健全区域15分钟养老服务圈。同时依托企业全产业链资源优势，持续深化公益+市场+社区合伙人模式，免费开展居民就业培训、落实养老消费券惠民政策，全力打造延安社区养老幸福家园特色标杆，助力宝塔区养老服务体系高质量发展，全面彰显民生养老服务保障实效。

## 八、平台功能设计与未来综合效果说明

### 8.1 建设目标

本项目同步搭建**社区服务智慧运营管理平台**（以下简称"平台"），以信息化手段支撑窑洞养老幸福家园的日常运营管理与服务交付。平台基于智慧居家养老服务管理平台成熟产品体系定制开发，聚焦社区养老场景下的老人健康管理、服务订单调度、服务人员管理、服务质量控制与数据可视化运营五大核心能力，实现"服务有记录、过程可追溯、质量有保障、数据可分析"的精细化运营目标。

### 8.2 功能模块设计

平台功能架构划分为**运营管理端**与**服务交付端**两大入口，覆盖从服务调度到效果评价的完整业务闭环。

#### 8.2.1 运营管理端

| 模块 | 功能说明 |
|------|----------|
| **首页工作台** | 聚合展示当日服务概况、待处理任务提醒、最近服务动态、服务质量评分等关键指标，支持快速跳转至各业务模块。管理员登录即可掌握机构运营全貌。 |
| **服务商管理** | 维护合作服务商基本信息、服务类型、服务范围、结算规则；按区域或服务类别筛选服务商，支持新增、编辑、资质上传与状态管理。 |
| **服务人员管理** | 统一管理护理员、康复师、医护等一线服务人员档案：基本信息、技能认证、健康状态、服务排班、紧急联系人；支持批量导入与证件管理。 |
| **老人档案管理** | 为每位入住或签约老人建立电子健康档案：基本信息、护理等级评估、慢病史、过敏史、紧急就医偏好；支持按区域、健康风险等级等维度筛选。 |
| **健康档案管理** | 接入健康监测设备（血压计、血糖仪、生命体征监测床垫等）推送数据，自动生成健康档案；支持健康报告生成、历史趋势查看、AI健康风险预警提示。 |
| **订单管理** | 全链路订单调度：服务预约 → 派单分配 → 服务人员接单 → 签到签出记录 → 服务完成确认 → 质检评价；支持订单状态实时追踪与超时提醒。 |
| **服务日志管理** | 服务人员每次上门服务须通过移动端填写服务日志：服务项目、服务内容、服务时长、现场照片（盲水印防篡改）、老人状态、异常情况说明；支持查看历史服务记录与服务统计。 |
| **合同管理** | 老人/家属与服务机构的电子合同签署与管理，支持腾讯电子签集成，实名认证签署、签署状态追踪、合同原文查看与下载归档。 |
| **服务评价管理** | 服务完成后自动触发满意度评价，支持星级评分+文字评价；机构可查看评价统计与明细，支持针对低分评价跟进整改。 |
| **质检管理** | 设立服务质检标准，支持按订单或服务日志开展质检检查，记录质检结果与整改要求；服务评价与质检结果共同纳入服务人员绩效考核。 |
| **财务管理** | 服务费用结算管理：订单金额核算、服务项目定价、优惠抵扣规则配置、账单生成与导出；支持服务商结算对账与机构收支分析。 |
| **系统管理** | 基础数据字典维护、角色权限配置、用户账号管理、操作日志审计；支持机构个性化参数配置。 |

#### 8.2.2 服务交付端（移动端H5）

| 模块 | 功能说明 |
|------|----------|
| **服务日历** | 服务人员查看个人当日及近期服务日程安排，按日期查看预约老人信息与服务地址，支持一键导航。 |
| **上门签到签出** | 服务人员到达老人住处后，通过移动端完成签到（定位+时间戳）；服务结束后填写服务日志并签出，系统自动记录服务时长。 |
| **服务日志填写** | 结构化填写服务内容、服务感受、老人状态、异常描述；支持现场拍照（自动添加时间、地点水印）；异常情况一键上报机构。 |
| **健康数据上报** | 对接智能健康监测设备数据推送，或由服务人员/老人手动录入当次健康测量结果（血压、血糖等），自动归档至老人健康档案。 |
| **订单处理** | 服务人员接收派单通知，确认接单或因故申请改派；服务完成后提交服务记录。 |

### 8.3 平台技术特性

- **多角色权限体系**：平台支持市级/区级政府监管账户、机构管理员账户、服务商账户、服务人员账户、老人/家属账户五级角色，各角色菜单与数据严格隔离，保障信息安全与业务隐私。
- **数据资产入表能力**：平台运营过程中积累的服务记录、健康档案、评价数据等，均以结构化形式存储于MySQL数据库，支持后续按需清洗、确权、评估入表，形成机构数据资产。
- **AI能力集成**：健康档案模块集成阿里百炼多模态大模型，支持健康报告智能解读、异常趋势预警、健康建议生成；服务日志图片支持AI质量审核，识别异常服务行为。
- **懒加载优化**：平台各业务模块（尤其是含大量图片的服务日志、健康档案页面）全面采用图片懒加载机制，页面首次加载时间控制在4秒以内，保障用户体验。
- **腾讯电子签集成**：合同签署全流程对接腾讯电子签SaaS API，实名认证→签署→回调归档全自动，支持查看任意状态合同原文。
- **公网免鉴权访问**：针对老人/家属预约等低敏感场景，平台提供免登录外部访问链接与二维码，支持微信扫码预约、满意度调查提交，便于推广落地。

### 8.4 实施计划

| 阶段 | 时间 | 主要工作 |
|------|------|---------|

| 第三阶段 | 第5–6月 | 试运营与服务日志全量上线、健康监测设备数据接入、数据看板调优、正式交付使用 |

### 8.5 预期综合效果

**运营效率提升**：平台将原有手工台账、电话调度、纸质签字的运营模式升级为全流程线上化管理，预计减少管理人员事务性工作量约40%，服务响应周期从平均48小时缩短至24小时内。

**服务质量可量化**：通过服务日志数字化记录、签到签出时间戳校验、现场照片留痕及质检闭环追踪，每单服务可还原服务全过程，质量投诉处理效率提升60%以上。

**数据驱动决策**：首页工作台实时聚合运营数据，机构管理者可随时掌握服务量、客户满意度、服务人员工作饱和度等关键指标，为服务优化与人员调度提供数据支撑。

**合规与风控**：腾讯电子签集成确保合同签署合法有效；操作日志全链路审计支持监管审查；健康数据异常预警机制可提前发现老人健康风险，降低意外事件发生率。

**模式可复制**：平台基于标准化模块设计，窑洞养老幸福家园运营模式验证成熟后，可快速复制至宝塔区乃至延安市其他社区养老机构，形成区域连锁效应，全面助力宝塔区15分钟养老服务圈建设目标如期实现。
