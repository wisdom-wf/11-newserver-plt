# 智慧居家养老服务管理平台

## 项目简介

**智慧居家养老服务管理平台** (Smart Home-Based Elderly Care Service Management Platform) 是一款面向政府监管和服务商管理的养老服务信息化解决方案。

### 背景

随着中国人口老龄化加剧，居家养老服务需求日益增长。本平台旨在建立一个统一、高效的养老服务管理体系，实现对养老服务的全流程监管。

### 系统定位

- **面向对象**: 市级/区县民政部门、街道/乡镇服务站、社区、养老服务提供商
- **核心目标**: 整合养老服务资源，提升服务质量，实现政府监管与市场化服务的有机结合

## 系统架构

### 技术栈

#### 后端 (Spring Boot)
- **框架**: Spring Boot 2.7+
- **ORM**: MyBatis-Plus 3.5+
- **数据库**: MySQL 8.0
- **认证**: JWT
- **构建**: Maven

#### 前端 (Vue3)
- **框架**: Vue 3.4+
- **构建**: Vite 5
- **UI库**: Naive UI
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **类型**: TypeScript 5
- **样式**: UnoCSS

## 快速开始

### 环境要求

- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.8+

### 数据库初始化

```bash
# 启动MySQL
docker start mysql-dev

# 创建数据库
docker exec mysql-dev mysql -u root -proot -e "CREATE DATABASE IF NOT EXISTS elderly_care CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 导入初始数据 (需要先执行建表SQL)
docker exec -i mysql-dev mysql -u root -proot elderly_care < sql/init.sql
```

### 后端启动

```bash
cd elderly-care-server

# 编译打包
mvn clean package -DskipTests

# 启动服务
java -jar target/elderly-care-server-1.0.0.jar

# 开发模式 (热部署)
mvn spring-boot:run
```

服务地址: http://localhost:8080

### 前端启动

```bash
cd dingfeng-work

# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建生产版本
pnpm build
```

访问地址: http://localhost:3000
