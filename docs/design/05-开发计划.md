# 智慧居家养老服务管理平台 - 开发计划

## 1. 项目概述

### 1.1 项目背景

智慧居家养老服务管理平台是面向政府民政部门、养老服务机构和老年人的综合性服务管理平台。系统实现养老服务的全流程数字化管理，包括服务商管理、服务人员管理、订单管理、老人客户管理、财务结算、服务评价等核心功能。

### 1.2 项目目标

| 目标类别 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 功能目标 | 完成8大子系统开发 | 8个子系统全部上线 |
| 性能目标 | 系统响应时间<500ms | TP99<500ms |
| 质量目标 | 测试覆盖率≥80% | 自动化测试覆盖 |
| 安全目标 | 通过等保三级认证 | 安全测评通过 |
| 交付目标 | 按期交付使用 | 2026-12-31上线 |

### 1.3 项目范围

#### 包含范围

- 服务商管理子系统
- 服务人员管理子系统
- 订单管理子系统
- 老人客户管理子系统
- 财务结算管理子系统
- 服务评估评价子系统
- 系统配置子系统
- 数据统计分析子系统

#### 不包含范围

- 智能硬件设备对接（预留接口）
- 第三方支付系统对接（预留接口）
- 民政系统数据对接（预留接口）

### 1.4 技术架构

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Vue.js 3 + Element Plus | PC端管理后台 |
| 后端 | Spring Boot 3 + Spring Cloud | 微服务架构 |
| 数据库 | MySQL 8.0 | 主数据库 |
| 缓存 | Redis 7.x | 缓存和会话 |
| 搜索引擎 | Elasticsearch 8.x | 全文搜索 |
| 消息队列 | RocketMQ 5.x | 异步消息 |
| 文件存储 | MinIO | 对象存储 |
| 容器 | Docker + Kubernetes | 容器化部署 |
