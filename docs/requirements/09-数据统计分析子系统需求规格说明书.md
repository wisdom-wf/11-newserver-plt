# 数据统计分析子系统需求规格说明书

## 1. 概述

### 1.1 子系统简介
数据统计分析子系统是智慧居家养老服务管理平台的数据中心，负责对平台各类业务数据进行统计分析，生成各类报表和图表，为政府决策和服务管理提供数据支撑。

### 1.2 业务目标
- 实现业务数据的全面统计分析
- 提供直观的数据可视化展示
- 支持多维度的数据分析挖掘
- 为决策提供数据支撑
- 提升数据管理水平和决策效率

## 2. 用户角色与权限

| 角色 | 权限说明 |
|------|----------|
| 市级管理员 | 全市数据查看、统计分析、报表生成、决策支持 |
| 区县管理员 | 本辖区数据查看、统计分析、报表生成 |
| 街道乡镇管理员 | 本辖区数据查看、统计分析 |
| 社区管理员 | 社区数据查看、统计分析 |
| 服务商管理员 | 本企业数据查看、统计分析 |
| 财务人员 | 财务数据查看、财务报表生成 |

## 4. 数据模型

### 4.1 统计报表配置表（t_stat_report_config）
```sql
CREATE TABLE t_stat_report_config (
    report_config_id VARCHAR(32) PRIMARY KEY COMMENT '报表配置ID',
    report_code VARCHAR(50) UNIQUE COMMENT '报表编码',
    report_name VARCHAR(100) COMMENT '报表名称',
    report_category VARCHAR(50) COMMENT '报表分类',
    report_desc VARCHAR(500) COMMENT '报表描述',
    data_source VARCHAR(100) COMMENT '数据源',
    dimensions TEXT COMMENT '统计维度',
    indicators TEXT COMMENT '统计指标',
    chart_type VARCHAR(20) COMMENT '图表类型',
    sort_order INT COMMENT '排序',
    status VARCHAR(20) COMMENT '状态',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 4.2 自定义报表表（t_custom_report）
```sql
CREATE TABLE t_custom_report (
    custom_report_id VARCHAR(32) PRIMARY KEY COMMENT '自定义报表ID',
    report_name VARCHAR(100) COMMENT '报表名称',
    report_desc VARCHAR(500) COMMENT '报表描述',
    data_source VARCHAR(100) COMMENT '数据源',
    dimensions TEXT COMMENT '统计维度',
    indicators TEXT COMMENT '统计指标',
    chart_type VARCHAR(20) COMMENT '图表类型',
    time_range VARCHAR(50) COMMENT '时间范围',
    creator_id VARCHAR(32) COMMENT '创建人ID',
    status VARCHAR(20) COMMENT '状态',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 4.3 导出记录表（t_export_record）
```sql
CREATE TABLE t_export_record (
    export_id VARCHAR(32) PRIMARY KEY COMMENT '导出ID',
    export_type VARCHAR(50) COMMENT '导出类型',
    export_format VARCHAR(20) COMMENT '导出格式',
    export_params TEXT COMMENT '导出参数',
    export_status VARCHAR(20) COMMENT '导出状态',
    export_url VARCHAR(500) COMMENT '导出文件地址',
    export_time DATETIME COMMENT '导出时间',
    export_user_id VARCHAR(32) COMMENT '导出人ID',
    file_size BIGINT COMMENT '文件大小',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_user (export_user_id)
);
```

### 4.4 统计数据表（t_stat_data）
```sql
CREATE TABLE t_stat_data (
    stat_data_id VARCHAR(32) PRIMARY KEY,
    report_code VARCHAR(50) COMMENT '报表编码',
    stat_date DATE COMMENT '统计日期',
    stat_type VARCHAR(50) COMMENT '统计类型',
    stat_dimension TEXT COMMENT '统计维度',
    stat_indicator VARCHAR(100) COMMENT '统计指标',
    stat_value DECIMAL(20,2) COMMENT '统计值',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_report_date (report_code, stat_date),
    INDEX idx_stat_date (stat_date)
);
```
