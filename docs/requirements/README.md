## 技术栈

### 前端技术
- Vue.js 3.x / React 18.x
- Element Plus / Ant Design
- Pinia / Redux Toolkit
- Vite / Webpack 5

### 后端技术
- Spring Boot 3.x
- Spring Cloud Alibaba
- MySQL 8.0 + Redis 7.x
- MyBatis Plus / JPA

### 基础设施
- Docker + Kubernetes
- RocketMQ / Kafka
- Elasticsearch
- MinIO / 阿里云OSS
