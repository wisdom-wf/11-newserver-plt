### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | #003375 | 深蓝，代表专业、科技 |
| 辅助色 | #1890FF | 亮蓝，代表智能、未来 |
| 强调色 | #52C41A | 绿色，代表增长、成功 |
| 警示色 | #FF4D4F | 红色，代表预警、紧急 |
| 背景色 | #F5F5F5 | 浅灰，内容区背景 |
| 文字色 | #262626 | 深灰，正文 |

### 字体建议

| 用途 | 字体 | 大小 |
|------|------|------|
| 标题 | 思源黑体/微软雅黑 | 36-44pt |
| 副标题 | 思源黑体/微软雅黑 | 24-28pt |
| 正文 | 思源黑体/微软雅黑 | 18-20pt |
| 数字 | Arial/Source Sans Pro | 28-36pt |

### 动画建议

- 标题出现：淡入，0.5秒
- 内容出现：从左滑入，0.3秒，依次延迟
- 数据强调：数字跳动效果
- 过渡：左右滑动

### 演示节奏建议

| 时间段 | 内容 | 时长 |
|--------|------|------|
| 0-2分钟 | 开场、目录 | 2分钟 |
| 2-5分钟 | 市场机遇 | 3分钟 |
| 5-7分钟 | 现有系统评估 | 2分钟 |
| 7-12分钟 | 升级方案介绍 | 5分钟 |
| 12-20分钟 | AI能力详解 | 8分钟 |
| 20-25分钟 | IoT集成 | 5分钟 |
| 25-30分钟 | 实施计划 | 5分钟 |
| 30-35分钟 | 投资回报 | 5分钟 |
| 35-40分钟 | 合作方式+QA | 5分钟 |

**总时长建议：35-40分钟（含QA）**
