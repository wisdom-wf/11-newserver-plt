### 健康档案 API 设计

```typescript
// 将来服务日志提交时可查询老人健康历史
GET /api/health-archive/elder/{elderId}/recent
  → 返回最近N次血压/血糖/用药记录（只读）

GET /api/health-archive/elder/{elderId}/summary
  → 返回健康画像：平均血压区间、常用药、慢性病标签

POST /api/health-archive/elder/{elderId}/ai-insight
  → AI分析建议（接入大模型）
```
