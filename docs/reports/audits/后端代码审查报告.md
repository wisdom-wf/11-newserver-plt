## 二、问题详情

### P0-1: Elder实体姓名字段命名错误

**文件**: `src/main/java/com/elderlycare/entity/elder/Elder.java`
**行号**: 约第28行

```java
// ❌ 错误写法
@TableField("elder_name")
private String name;

// ✅ 修复方案
@TableField("elder_name")
private String elderName;
```

**影响**: MyBatis-Plus查询时elder_name无法正确映射到name字段，导致老人姓名显示为null

## 五、快速修复脚本

如需立即修复P0问题，可执行以下修改：

```bash
# 1. Elder.java - name改为elderName
sed -i '' 's/private String name;/private String elderName;/g' \
  src/main/java/com/elderlycare/entity/elder/Elder.java

# 2. ElderMapper.xml - property name改为elderName
sed -i '' 's/property="name"/property="elderName"/g' \
  resources/mapper/elder/ElderMapper.xml

# 3. CreateOrderDTO.java - 添加providerId
# 需手动添加字段

# 4. OrderController.java - receive改为accept
sed -i '' 's/@PutMapping(\"\/{orderId}\/receive\")/@PutMapping(\"\/{orderId}\/accept\")/g' \
  src/main/java/com/elderlycare/controller/order/OrderController.java
```
