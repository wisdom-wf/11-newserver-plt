## 🔴 P0 严重问题

### 问题1: 订单状态枚举未定义

**文件**: `src/typings/api/order.d.ts`

**问题描述**:

- 前端 `Order` 接口的 `status` 字段定义为 `string`
- 后端存在完整的 `OrderStatus` 枚举，但前端未定义对应类型

**后端枚举值**:

```java
CANCELLED("CANCELLED", "已取消", 0)
CREATED("CREATED", "待派单", 2)        // 注意：没有1
DISPATCHED("DISPATCHED", "已派单", 3)
RECEIVED("RECEIVED", "已接单", 4)
SERVICE_STARTED("SERVICE_STARTED", "服务中", 5)
SERVICE_COMPLETED("SERVICE_COMPLETED", "已完成", 6)
EVALUATED("EVALUATED", "已评价", 7)
SETTLED("SETTLED", "已结算", 8)
REJECTED("REJECTED", "已拒单", 9)
```

**修复建议**:

```typescript
// 在 order.d.ts 中添加
type OrderStatus =
  | 'CANCELLED' // 已取消
  | 'CREATED' // 待派单
  | 'DISPATCHED' // 已派单
  | 'RECEIVED' // 已接单
  | 'SERVICE_STARTED' // 服务中
  | 'SERVICE_COMPLETED' // 已完成
  | 'EVALUATED' // 已评价
  | 'SETTLED' // 已结算
  | 'REJECTED'; // 已拒单
```
