## 修复 useNaivePaginatedTable Hook 的 api 参数问题

### 问题

### 原因
`useNaivePaginatedTable` hook 接受 `apiFn` 参数，但没有将其转换为 `api` 函数传递给底层的 `useTable` hook。

### 修复
修改文件：`src/hooks/common/table.ts`

1. 从 options 中解构出 `apiFn` 和 `apiParams`
2. 创建 `api` 函数包装 `apiFn`
3. 将 `api` 传递给 `useTable`

```typescript
// Create api function from apiFn
const api = async () => {
  if (options.apiFn) {
    return options.apiFn(paginationParams.value);
  }
  throw new Error('apiFn is required');
};

// Destructure to exclude apiFn and apiParams from options passed to useTable
const { apiFn: _apiFn, apiParams: _apiParams, ...tableOptions } = options;

const result = useTable<ResponseData, ApiData, NaiveUI.TableColumn<ApiData>, true>({
  ...tableOptions,
  api,  // 添加 api 参数
  // ...
});
```

## 修复服务人员页面编辑功能

### 问题
编辑服务人员时，表单没有回填旧数据。

### 原因
缺少 `watch` 监听 `editingData` 变化来回填表单数据。

### 修复文件
`src/views/business/staff/index.vue`

### 修复内容
1. 添加 `resetForm()` 函数清空表单
2. 添加 `watch` 监听 `editingData`，在编辑时自动回填数据

```typescript
// Reset form to empty state
function resetForm() {
  form.value = {
    staffName: '',
    gender: 0,
    idCard: '',
    phone: '',
    providerId: '',
    emergencyContact: '',
    emergencyPhone: '',
    remark: '',
    status: 'PENDING'
  };
}

// Watch editingData to fill form when editing
watch(
  () => editingData.value,
  (data) => {
    if (data) {
      form.value = {
        staffName: data.staffName || '',
        gender: data.gender ?? 0,
        idCard: data.idCard || '',
        phone: data.phone || '',
        providerId: data.providerId || '',
        emergencyContact: data.emergencyContact || '',
        emergencyPhone: data.emergencyPhone || '',
        remark: data.remark || '',
        status: (data.status as 'PENDING' | 'ON_JOB' | 'OFF_JOB') || 'PENDING'
      };
    } else {
      resetForm();
    }
  },
  { immediate: true }
);
```

## 修复预约确认服务商选择 - 改用列表接口

```typescript
export async function fetchGetProviderOptions(params?: { areaId?: string; serviceCategory?: string }) {
  const res = await fetchGetProviderList({ ...params, page: 1, size: 1000 });
  return (res.data?.records || []).map(item => ({
    id: item.providerId,
    name: item.providerName
  }));
}
```
